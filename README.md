# Dr. Marcelo Melo - Advocacia Especializada

Site institucional desenvolvido para o escritório Dr. Marcelo Melo, especializado em Direito Previdenciário, Trabalhista, Consumidor e Cível.

## Executar Localmente

**Pré-requisitos:** Node.js

1. Instale as dependências:
   `npm install`

2. Execute o projeto:
   `npm run dev`
